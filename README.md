# 🎯 Problema de Negócio

A diretoria não possuía uma visão consolidada das operações globais.

Os dados estavam:

- Separados por país
- Em arquivos CSV distintos
- Sem padronização
- Com inconsistências e categorias inválidas

O objetivo foi criar uma **fonte única de verdade**, gerar métricas financeiras 
confiáveis e permitir decisões baseadas em dados.

# 🧠 Engenharia de Dados


# 🧹 Data Cleaning Estratégico

Problemas encontrados:

- Categoria inválida chamada "Again"
- Produtos com nomes genéricos
- Valores nulos em preço e quantidade

Decisões aplicadas:

- Imputação estatística (média e moda)
- Padronização de categorias
- Remoção de inconsistências que comprometeriam análises executivas

A limpeza foi conduzida com visão de impacto financeiro — não apenas técnica.
